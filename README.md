# c_matrix

Библиотека на C для базовых операций с матрицами. Реализовано всё, что нужно для работы с прямоугольными матрицами произвольного размера: создание, удаление, сравнение, сложение, вычитание, умножение (на число и на другую матрицу), транспонирование, вычисление определителя, матрицы алгебраических дополнений и обратной матрицы.

Матрицы хранятся в структуре:

```c
typedef struct matrix_struct {
    double** matrix;
    int rows;
    int columns;
} matrix_t;
```

Элементы — `double`, сравнение идёт с точностью до `1e-6`.

Проект из Школы 21, написан строго по заданию: C11, Google Style, структурное программирование, никаких устаревших фич.

## Что умеет

- `s21_create_matrix` — создать матрицу
- `s21_remove_matrix` — освободить память
- `s21_eq_matrix` — сравнить две матрицы
- `s21_sum_matrix` / `s21_sub_matrix` — сложение и вычитание
- `s21_mult_number` — умножение на скаляр
- `s21_mult_matrix` — матричное умножение
- `s21_transpose` — транспонировать
- `s21_calc_complements` — матрица алгебраических дополнений
- `s21_determinant` — определитель (только для квадратных)
- `s21_inverse_matrix` — обратная матрица (если определитель ≠ 0)

Все функции (кроме сравнения и удаления) возвращают код:

- `0` — успех
- `1` — некорректная матрица
- `2` — ошибка вычисления

## Сборка и тесты

Всё через Makefile:

- `make` — собирает статическую библиотеку `s21_matrix.a`
- `make test` — собирает и запускает тесты (Check)
- `make gcov_report` — отчёт по покрытию в HTML (покрытие по каждой функции >80%)
- `make clean` — убирает всё лишнее

Тестов хватает: проверяют обычные случаи, граничные значения, разные размеры, ошибки инициализации и вычислений. При запуске видно всё подробно.

## Как пользоваться

Подключаешь `s21_matrix.h` и линкуешь библиотеку.

### Пример

```c
#include "s21_matrix.h"

int main() {
    matrix_t A, B, result;
    s21_create_matrix(2, 2, &A);
    s21_create_matrix(2, 2, &B);

    A.matrix[0][0] = 1; A.matrix[0][1] = 2;
    A.matrix[1][0] = 3; A.matrix[1][1] = 4;

    B.matrix[0][0] = 5; B.matrix[0][1] = 6;
    B.matrix[1][0] = 7; B.matrix[1][1] = 8;

    s21_sum_matrix(&A, &B, &result);
    // result теперь содержит сумму

    s21_remove_matrix(&A);
    s21_remove_matrix(&B);
    s21_remove_matrix(&result);

    return 0;
}
```
